# How to Run (AMPL)

## 1. Install AMPL
- Download AMPL from: https://ampl.com
- Install **AMPL Community / Student Edition**
- Check installation:
```bash
ampl
```

## 2. Run the Model
- Go to the project directory and run:
```bash
ampl run.run
```
AMPL will load the model and data files, solve the optimization problem, and print the results.
